# AUTO|Q AI Test Expert Knowledge Base

This repository contains the knowledge base used to ground the AUTO|Q AI Test Expert agent.

The knowledge base is organized into:

## Method Knowledge
Professional testing methodology based on ISTQB and quality engineering practices.

## Response Frameworks
Structured reasoning guides used by the AI agent to generate high-quality testing outputs.

The goal of this repository is to enable an AI agent that can support consultants with:

- requirement analysis
- test condition derivation
- test scenario generation
- test case generation
- test strategy advice
